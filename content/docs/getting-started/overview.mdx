# Overview

Q2Labs is an agentic research tool designed specifically for the healthcare market. It reads your unstructured data, runs complex quantitative analysis, and integrates directly with your existing secure vault.

<Callout type="info">
  **Early Access Note:** The Q2Labs CLI and Vault integration are currently in beta. Ensure your environment meets the strict compliance standards before processing PHI (Protected Health Information).
</Callout>

## What you can do

Here are some of the ways you can use Q2Labs to accelerate your workflows:

* **Qualitative Field Analysis:** Automatically transcribe and extract key themes from patient interviews.
* **Sovereign Data Processing:** Keep all sensitive data strictly within your local environment.
* **Automated Reporting:** Generate compliant research summaries with a single command.

## Quick Install

To install the Q2Labs CLI securely in your environment, use one of the following methods depending on your operating system:

<CodeTabs>
  <Tab title="macOS / Linux">
    ```bash
    curl -fsSL [https://q2labs.ai/install.sh](https://q2labs.ai/install.sh) | bash
    ```
  </Tab>
  <Tab title="Windows">
    ```powershell
    irm [https://q2labs.ai/install.ps1](https://q2labs.ai/install.ps1) | iex
    ```
  </Tab>
</CodeTabs>

## Next Steps

Once installed, you'll need to authenticate your machine with your Q2Labs API key. Check out the [Quickstart guide](./quickstart.mdx) to run your first analysis.
